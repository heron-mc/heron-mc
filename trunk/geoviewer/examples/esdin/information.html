This is the Data Viewer for the ESDIN project. It can display data from Download Services and View Services.

Data from View Services can be selected in the "Base layers" node of the map layer tree. Data from Download Services are available from the other nodes, which are grouped according to ESDIN theme (some INSPIRE Annex I themes plus ExM). Below the level of ESDIN themes there are the feature types that are defined in the data specifications for the themes.

If there is at least one configured data provider for a feature type of a Download Service, the feature type will have a check box, enabling the downloading of features.

For discussion on this viewer please use EuroGeoForum.

The ESDIN Data Viewer is an example application of the geoext-viewer, which is an Open Source project. Anyone interested to contribute to the project is welcome.
