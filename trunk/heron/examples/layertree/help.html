About this Example

This example shows how to create an explicit Layer tree. The legends in the LayerTree are added in one of two ways.

  • By explicitly setting the 'text' property of the 'gx_layer' tree node with GFI Image + text (USA States WMS)
  • By setting the 'legend: true' property of the 'gx_layer' tree node. This results in a full legend panel under the text ('Full' examples and USA States WFS).
