About this Example

Shows WMS Time interaction via Date Slider. Integrated in Heron as User Extension (ux).

Usage

Slide the date between mid 2013 and the beginning of 2014 to see occurrences of earthquakes in The Netherlands. Use Identify (I button) to get detailed info.

  • Move timeline: Point to timeline and drag to left or right.
  • Move filter period: Point to yellow period and drag to left or right.
  • Move start/end date: Point to grey bar and drag to left or right.
  • Zoom in/out: zoom in or out to change the scale of the timeline (only when enabled through config)

Configuration

See Config.js (xtype: hr_timesliderpanel) and API section in TimeSliderPanel.js, further standard NL default options.

Source Data

The example should work with the layer available as public WMS as specified in config (no further work needed)

Data for WMS was obtained and configured a follows:

  • Source data from http://www.knmi.nl/seismologie/aardbevingen-nederland.html.
  • Downloaded as CSV.
  • Minor bulk edit CSV: shorter field names, "Datum" and removed time from Datum field
  • Converted CSV with GDAL to Shapefile (applied .csvt and VRT to generate real Date and Point attributes).
  • Shapefile added to GeoServer: under Layer "Dimensions" tab, add "Datum field" as "continuous
  • Download Shapefile here
